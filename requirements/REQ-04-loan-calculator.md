# REQ-04: 킬러 계산기 #2 - 주택대출 종합 시뮬레이터

**대응 주차**: Week 4
**담당**: FE, UX/UI
**PRD 참조**: 섹션 3.1.1
**URL**: `/finance/loan-calculator`

---

## 1. 목적

다양한 상환 방식을 한눈에 비교하고, 시중 금리 데이터와 연동하여 대출 의사결정을 지원하는 킬러 계산기를 구현한다.

---

## 2. 요구사항

### 2.1 입력

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-04-01 | 매매가 입력 (슬라이더 + 직접 입력, 범위: 1,000만~30억) | 필수 | 슬라이더 양방향 동기화 |
| R-04-02 | 자기자금 비율 슬라이더 (0~100%) 또는 대출금액 직접 입력 전환 | 필수 | 비율 변경 시 대출금액 자동 계산, 반대도 동일 |
| R-04-03 | 연이자율: 직접 입력 + 금리 프리셋 드롭다운 | 필수 | 프리셋 선택 시 이자율 자동 입력 |
| R-04-04 | 금리 프리셋: "시중 평균 금리 적용" 버튼 + 주요 은행별 금리 목록 | 필수 | 빌드 타임 fetch 데이터 사용, 데이터 출처/갱신일 표시 |
| R-04-05 | 대출기간 선택 (1~40년, 기본값: 30년) | 필수 | 드롭다운 또는 슬라이더 |
| R-04-06 | 거치기간 선택 (0~10년, 기본값: 0) | 필수 | 대출기간 이내로 제한 |
| R-04-07 | 연봉 입력 (선택, DSR 계산용) | 권장 | 입력 시 DSR 섹션 활성화 |
| R-04-08 | URL 쿼리 파라미터로 초기값 설정 (`?price=500000000&rate=3.5&annual=50000000` 등) | 필수 | 연봉 분석기에서 넘어온 값 자동 입력 |

### 2.2 계산 로직

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-04-09 | 원리금균등 상환 계산 (거치기간 포함) | 필수 | 월 상환액, 총 상환액, 총 이자 정확 |
| R-04-10 | 원금균등 상환 계산 (거치기간 포함) | 필수 | 월별 상환액 변동 반영 |
| R-04-11 | 만기일시 상환 계산 | 필수 | 매월 이자만, 만기 시 원금 일시 상환 |
| R-04-12 | 월별 상환 스케줄 생성 (원금, 이자, 잔액) | 필수 | 전체 대출기간에 대한 월별 데이터 배열 |
| R-04-13 | DSR 계산: 연간 원리금 상환액 / 연봉 × 100 | 필수 | 연봉 입력 시 자동 계산 |
| R-04-14 | 계산 로직은 순수 함수로 분리 (`lib/calculators/loan.ts`) | 필수 | 단위 테스트 가능 |

### 2.3 출력

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-04-15 | 핵심 요약 카드: 월 상환액(원리금균등 기준), 총 상환액, 총 이자 | 필수 | 결과 최상단, 강조 표시 |
| R-04-16 | 상환방식 비교 테이블: 원리금균등/원금균등/만기일시 3열 비교 (월 상환액, 총 상환액, 총 이자) | 필수 | 한눈에 비교 가능 |
| R-04-17 | DSR 표시: 연봉 대비 상환 부담률 (연봉 입력 시만 표시) | 권장 | 40% 초과 시 경고 색상(Red) 표시 |

### 2.4 시각화

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-04-18 | 상환방식 비교 바 차트: 3개 방식의 총 이자 비교 | 필수 | 색상으로 방식 구분 |
| R-04-19 | 월별 원리금 스택 바 차트: 원금(Blue)과 이자(Light Blue) 색상 구분 | 필수 | 원리금균등 기준, 시간 흐름에 따른 변화 시각화 |
| R-04-20 | 입력 변경 시 모든 차트 실시간 갱신 | 필수 | 버튼 없이 즉시 반영 |
