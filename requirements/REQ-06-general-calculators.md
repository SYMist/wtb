# REQ-06: 일반 계산기 1차 배치 (7종)

**대응 주차**: Week 6
**담당**: FE
**PRD 참조**: 섹션 3.2

---

## 1. 목적

공통 템플릿(REQ-02)을 활용하여 킬러 계산기와 직접 연결되는 일반 계산기 7종을 빠르게 구현한다. Phase 1 런칭 시 총 10종(킬러 3 + 일반 7)의 계산기를 확보한다.

---

## 2. 공통 요구사항

모든 일반 계산기에 적용되는 공통 사항:

| ID | 요구사항 | 인수 조건 |
|---|---|---|
| R-06-00-A | 일반 계산기 레이아웃 템플릿 사용 (REQ-02 R-02-12) | 단일 컬럼 + 사이드바 |
| R-06-00-B | 입력 변경 시 결과 즉시 갱신 (계산 버튼 없음) | |
| R-06-00-C | URL 쿼리 파라미터로 입력값 초기화/공유 가능 | |
| R-06-00-D | 결과 공유 (링크 복사 + 카카오톡) | |
| R-06-00-E | 하단 가이드 텍스트 (접힌 상태, 300~500자) | 마케터 콘텐츠 삽입 |
| R-06-00-F | 하단 관련 계산기 카드 3~4개 | |
| R-06-00-G | 고유 title, h1, meta description, JSON-LD FAQ | |
| R-06-00-H | 계산 로직은 `lib/calculators/` 하위에 순수 함수로 분리 | |

## 3. 개별 계산기 요구사항

### 3.1 예적금 이자 계산기

**URL**: `/finance/deposit-calculator`
**연결**: 투자 복리 계산기

| ID | 요구사항 | 우선순위 |
|---|---|---|
| R-06-01 | 입력: 예치금액, 이자율(연), 기간(개월), 상품유형(정기예금/적금), 이자과세(일반/비과세/세금우대) | 필수 |
| R-06-02 | 출력: 세전이자, 세후이자, 만기수령액, 적용세율 | 필수 |
| R-06-03 | 적금: 단리/월복리 선택 | 필수 |
| R-06-04 | 이자과세: 일반(15.4%), 비과세(0%), 세금우대(9.5%) | 필수 |
| R-06-05 | title: "예적금 이자 계산기 - 세후 수령액 계산 \| Tooly" | 필수 |

### 3.2 전월세 전환 계산기

**URL**: `/finance/rent-conversion`
**연결**: 주택대출 시뮬레이터

| ID | 요구사항 | 우선순위 |
|---|---|---|
| R-06-06 | 입력: 보증금, 월세, 전환율(기본값: 법정 전환율) | 필수 |
| R-06-07 | 출력: 전세 환산 보증금 또는 월세 환산액 (양방향 변환) | 필수 |
| R-06-08 | 법정 전환율 자동 반영 (한국은행 기준금리 + 대통령령 가산율) | 필수 |
| R-06-09 | title: "전월세 전환 계산기 - 보증금 월세 변환 \| Tooly" | 필수 |

### 3.3 퇴직금 계산기

**URL**: `/finance/severance-calculator`
**연결**: 연봉 실수령액 분석기

| ID | 요구사항 | 우선순위 |
|---|---|---|
| R-06-10 | 입력: 입사일, 퇴사일, 최근 3개월 월급(또는 연봉에서 자동 계산) | 필수 |
| R-06-11 | 출력: 퇴직금 총액, 근속기간, 1일 평균임금 | 필수 |
| R-06-12 | 퇴직금 = 1일 평균임금 × 30일 × (근속일수/365) | 필수 |
| R-06-13 | title: "퇴직금 계산기 - 예상 퇴직금 조회 \| Tooly" | 필수 |

### 3.4 양도소득세 계산기

**URL**: `/finance/capital-gains-tax`
**연결**: 주택대출 시뮬레이터

| ID | 요구사항 | 우선순위 |
|---|---|---|
| R-06-14 | 입력: 취득가액, 양도가액, 보유기간, 1세대1주택 여부, 필요경비 | 필수 |
| R-06-15 | 출력: 양도차익, 장기보유 특별공제, 과세표준, 세율, 양도소득세, 지방소득세 | 필수 |
| R-06-16 | 장기보유 특별공제율 자동 적용 (보유기간별) | 필수 |
| R-06-17 | 1세대1주택 비과세 조건 안내 | 권장 |
| R-06-18 | title: "양도소득세 계산기 - 부동산 양도세 자동 계산 \| Tooly" | 필수 |

### 3.6 평수 ↔ 제곱미터 변환기

**URL**: `/convert/area-converter`
