# REQ-02: 공통 컴포넌트 및 레이아웃

**대응 주차**: Week 2
**담당**: FE, BE, 마케터
**PRD 참조**: 섹션 3.3, 4, 5.5, 5.6, 6.1, 7

---

## 1. 목적

모든 페이지에서 재사용되는 공통 컴포넌트와 레이아웃을 구현한다. 이후 계산기 개발 시 이 컴포넌트들을 조합하여 빠르게 페이지를 구성할 수 있는 기반을 만든다.

---

## 2. 요구사항

### 2.1 GNB (Global Navigation Bar)

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-02-01 | 좌측 로고 (Tooly), 중앙 카테고리 네비게이션 (금융/건강/변환/날짜/생활), 우측 검색 아이콘 | 필수 | 모든 페이지에서 동일하게 표시 |
| R-02-02 | 카테고리 호버 시 드롭다운으로 해당 카테고리의 계산기 목록 표시 | 필수 | 마우스 호버 시 드롭다운 노출 |
| R-02-03 | 모바일: 햄버거 메뉴로 전환, 터치 시 전체 메뉴 표시 | 필수 | 768px 이하에서 햄버거 메뉴 동작 |
| R-02-04 | 현재 페이지 카테고리 하이라이트 | 권장 | 활성 카테고리에 시각적 표시 |

### 2.2 푸터

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-02-05 | 사이트 소개, 개인정보처리방침, 데이터 출처 고지 링크 | 필수 | 링크 클릭 시 해당 페이지 이동 |
| R-02-06 | 카테고리별 주요 계산기 링크 (사이트맵 역할) | 권장 | 내부 링크 SEO 강화 |

### 2.3 검색 기능

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-02-07 | 홈페이지: 히어로 영역에 큰 검색바 ("무엇을 계산할까요?" 플레이스홀더) | 필수 | 홈 진입 시 검색바가 중앙에 크게 표시 |
| R-02-08 | 전체 페이지: GNB 검색 아이콘 클릭 시 검색 모달 또는 확장 | 필수 | 모든 페이지에서 검색 접근 가능 |
| R-02-09 | 자동완성: 키워드 입력 시 관련 계산기 이름/키워드 매칭하여 드롭다운 | 필수 | "전세" 입력 시 "전월세 전환 계산기" 등 노출 |
| R-02-10 | 검색 데이터: 계산기 이름, 카테고리, 관련 키워드를 정적 데이터로 관리 | 필수 | 설정 파일에서 검색 데이터 관리 |

### 2.4 계산기 공통 템플릿

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-02-11 | **킬러 계산기 레이아웃**: 데스크톱 좌우 분할 (좌: 입력폼 / 우: 결과 sticky), 모바일 상하 배치 | 필수 | 브레이크포인트 기준 레이아웃 전환 |
| R-02-12 | **일반 계산기 레이아웃**: 단일 컬럼 (입력 → 결과 → 광고 → 가이드 → 관련 계산기), 데스크톱 사이드바 | 필수 | 단일 컬럼 + 사이드바 레이아웃 |
| R-02-13 | 결과 영역 sticky 동작 (데스크톱, 킬러 계산기) | 필수 | 입력폼 스크롤 시 결과 영역 고정, 높이 초과 시 내부 스크롤 |
| R-02-14 | 가이드 텍스트 접기/펼치기 컴포넌트 | 필수 | 기본 접힌 상태, 클릭 시 펼침. HTML에는 항상 포함 |
| R-02-15 | 관련 계산기 카드 섹션 ("함께 많이 사용하는 계산기") | 필수 | 3~4개 카드 그리드, 설정 파일에서 관계 정의 |
| R-02-16 | 컨텍스트 CTA 버튼 컴포넌트 | 필수 | 텍스트와 대상 URL(쿼리 파라미터 포함)을 props로 받음 |

### 2.5 입력 폼 컴포넌트

| ID | 요구사항 | 우선순위 | 인수 조건 |
|---|---|---|---|
| R-02-17 | 숫자 입력 필드 (천 단위 콤마 자동 포맷, 슬라이더 연동 옵션) | 필수 | 입력 시 콤마 표시, 슬라이더와 양방향 동기화 |
| R-02-18 | 슬라이더 컴포넌트 (min/max/step 설정, 현재 값 표시) | 필수 | 드래그 시 값 변경, 접근성 기준 터치 영역 44px 이상 |
| R-02-19 | 셀렉트 드롭다운 | 필수 | 옵션 선택 가능 |
| R-02-20 | 입력 그룹 (소제목 + 입력 항목들) | 필수 | 시각적 그룹핑 |
| R-02-21 | 모든 입력 변경 시 결과 즉시 갱신 (계산 버튼 없음) | 필수 | 입력값 변경 즉시 결과 영역 업데이트 |
