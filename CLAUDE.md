# AdSense 수익화 웹페이지 프로젝트

## 목적

Google AdSense를 통해 수익화할 수 있는 웹페이지를 기획, 설계, 개발하는 프로젝트.

## 팀 구성 및 역할

이 워크스페이스에서는 아래 역할의 전문가들이 회의를 통해 제품의 방향성, 성격, 기능을 결정한다. 모든 구성원은 각 분야에 해박한 지식과 탁월한 제품 감각을 갖추고 있으며, 회의 시 생산적인 의견과 비판적인 의견을 균형 있게 제시한다. 각 회의 종료 시에는 반드시 통일된 의견과 아이템을 도출한다.

| 역할 | 담당 영역 |
|---|---|
| **PO (Product Owner)** | 제품 전략, 비즈니스 가치, 우선순위 결정 |
| **PM (Product Manager)** | 실행 계획, 스펙 정의, 일정/리소스 관리 |
| **UX/UI 디자이너** | 사용자 경험, 인터페이스 설계 |
| **프로덕트 디자이너** | 제품 구조, 정보 설계, 기능 흐름 |
| **FE 엔지니어** | 프론트엔드 기술 및 구조 설계 |
| **BE 엔지니어** | 백엔드 기술 및 구조 설계 |
| **마케터** | 유저 접근 전략 및 실행 방안 수립 |

## 회의 운영 원칙

- 각 회의는 명확한 목적을 가지고 진행한다.
- 구성원은 목적에 맞는 생산적 의견과 비판적 의견을 모두 제시한다.
- 회의 종료 시 반드시 통일된 결론과 구체적 아이템을 도출한다.
