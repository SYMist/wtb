# 네이버 블로그 플레이북 (Tooly 백링크용)

Tooly(`tooly.deluxo.co.kr`)로 검색 유입과 백링크를 확보하기 위한 네이버 블로그 운영 가이드입니다. 포스팅 5~10개를 기준으로 작성되었습니다.

---

## 1. 목표와 전략

### 핵심 KPI
- **직접 유입**: 블로그 → Tooly CTA 클릭률 **10% 이상**
- **검색 노출**: 목표 키워드 각 포스팅당 최소 1개 상위 10위권 진입
- **도메인 권위**: 네이버 블로그 자체의 C-Rank 지수 누적 (장기)

### 네이버 검색 알고리즘 대응 포인트
네이버는 Google과 다르게 **C-Rank + DIA** 알고리즘을 씁니다.

| 요소 | 의미 | 실무 적용 |
|---|---|---|
| **C-Rank (Context)** | 출처·인용의 신뢰도 | 공식 출처(국세청, 한국은행, 통계청) 명시 |
| **C-Rank (Chain)** | 주제 일관성 | 한 블로그는 금융/생활정보 장르로 고정 |
| **C-Rank (Creation)** | 활동성 | 주 1~2회 꾸준한 발행 |
| **DIA (체류시간)** | 독자 만족도 | 목차·이미지·표로 스크롤 유도 |
| **DIA (정보성)** | 실질적 도움 | 계산 예시·수치·근거 포함 |

### 주의사항
- **외부 링크 남발 금지**: 한 포스팅당 Tooly 링크는 **1~2개**만 (자연스러운 맥락에서)
- **복붙 금지**: 각 포스팅 최소 70% 이상 original text
- **이미지 5~10장**: 네이버는 이미지 많은 글을 선호 (Tooly 결과 스크린샷 적극 활용)
- **제목에 숫자·년도**: "2026년", "5분 만에", "얼마인지"

## 2. 주제 선정 원칙

### 좋은 주제 = 아래 3가지 교집합
1. **검색 수요**: 네이버 검색량 있는 질문형 키워드
2. **Tooly 연결성**: Tooly 계산기로 답변 가능
3. **해설 가치**: 계산기만으론 부족한 맥락·예시·팁 제공 필요

### 키워드 리서치 도구
- 네이버 광고 **키워드 도구** (검색량 확인, 무료)
- 네이버 **자동완성** (연관 질문 확인)
- **블랙키위** / **키워드마스터** (네이버 특화)

```markdown

- 핵심 답변 1~3줄
- 예: "연봉 5000만원이면 월 실수령액 약 **354만원**입니다. (2026년 기준, 부양가족 1인)"

## 1. [주제 개념 설명] (300~500자)

- **Tooly 결과 스크린샷** 1~2장 삽입

- 자연스러운 CTA:
  > 직접 해보고 싶다면 [Tooly 연봉 계산기](https://tooly.deluxo.co.kr/finance/salary-calculator)에서 숫자만 바꿔 확인할 수 있습니다.

- 핵심 요약 1줄
```

## 4. 작성 체크리스트

발행 전 반드시 확인:

### 제목
- [ ] 30자 내외 (너무 길면 검색 결과에서 잘림)
- [ ] 핵심 키워드 1개 + 수식어(년도/숫자/질문)
- [ ] 클릭 유도하는 질문형/숫자형
- [ ] 낚시성 금지 (예: "충격", "대박" 등)

### 본문
- [ ] 1,500자 이상 (체류시간 확보)
- [ ] 이미지 5장 이상 (결과 스크린샷·표·그래프)
- [ ] 목차(H2) 3개 이상
- [ ] 표(테이블) 1개 이상
- [ ] 공식/수치에 **출처** 명시
- [ ] Tooly 외부링크 **1~2개** (과다 금지)

### SEO
- [ ] 대표 키워드가 제목·첫 문단·소제목·마지막 문단에 분산 배치
- [ ] 연관 키워드 3~5개가 자연스럽게 본문에 등장
- [ ] 태그(해시태그) 5~10개 (대표 키워드 + 연관어)

### 발행 후
- [ ] 발행 직후 본인 SNS/커뮤니티에 1회 공유 (초기 노출 확보)
- [ ] 댓글 달리면 24시간 내 응답 (활동성 점수)
- [ ] 1주일 후 네이버 검색에서 "제목 일부" 검색해 노출 확인

## 6. 측정

발행 후 2주 간격으로 확인:

1. **Tooly GA4**: `utm_source=naver_blog&utm_medium=referral` 파라미터 붙여서 각 블로그 링크 걸기 → 유입량 추적
2. **네이버 블로그 통계**: 유입 검색어, 체류시간
3. **네이버 검색**: 타겟 키워드 순위 변화

## 부록. 첫 포스팅 샘플 오프닝 (1번 연봉 5000)

> **"연봉 5000 받는데 실수령액은 왜 이것밖에 안 되지?"** 한 번쯤 통장을 보며 생각해본 적 있을 거예요.
>
> 연봉 5000만원의 세전·세후 차이는 약 **천만원**에 달합니다. 소득세, 4대 보험, 부양가족 공제 여부에 따라 월 실수령액이 크게 달라지는데요. 오늘은 2026년 기준으로 **연봉 5000만원 실수령액**을 정확하게 계산하고, 세전·세후 구조를 쉽게 정리해드릴게요.
>
> 마지막에는 본인 연봉을 직접 넣어볼 수 있는 계산기도 소개합니다.
>
> ## 🔑 결론부터
> - 연봉 5000만원 → **월 실수령액 약 354만원** (부양가족 1인, 2026년 기준)
> - 연간 공제 총액 약 **754만원** (소득세 + 지방세 + 4대보험)
> - 비과세 식대 20만원 포함 시 실수령액 +약 3만원 증가
