# 네이버 블로그 플레이북 (Tooly 도달·인지 채널)

Tooly(`tooly.deluxo.co.kr`)로 **검색 유입·브랜드 인지**를 확보하기 위한 네이버 블로그 운영 가이드입니다. 포스팅 5~10개를 기준으로 작성되었습니다.

> ⚠️ **2026-05-30 역할 재정의**: 이 블로그는 Tooly의 **메인 유입 깔때기가 아니라 도달·인지용 보조 채널**입니다. 자연 검색(롱테일 시나리오 SEO)이 본체이고, 블로그는 초기 인지와 약한 트래픽 신호를 보탤 뿐입니다. 홈판 트래픽 = 저의도이므로 Tooly 전환을 과신하지 마세요.

> ⚠️ **"백링크" 오해 교정**: 네이버 블로그의 외부 링크는 거의 전부 **`nofollow`**라, **구글 검색 권위(PageRank)를 Tooly로 넘겨주지 않습니다.** 즉 이 블로그의 Tooly 기여는 *SEO 백링크가 아니라* ① 직접 클릭 유입(referral), ② 브랜드 인지, ③ 사람들이 "Tooly"를 검색·언급하게 만드는 간접 신호입니다. 이 전제로 운영 목표를 잡으세요.

## 1. 목표와 전략

### 핵심 KPI
- **직접 유입**: 블로그 → Tooly CTA 클릭률 **10% 이상** (referral, 백링크 아님)
- **검색 노출**: 목표 **롱테일** 키워드 각 포스팅당 최소 1개 상위 노출(통합검색 스마트블록 진입)
- **블로그 지수 누적**: 네이버 C-Rank/DIA+ 지수 (장기). 저품질 0건 유지가 최우선

### 네이버 검색 알고리즘 대응 포인트
네이버는 Google과 다르게 **C-Rank + DIA+(다이아 플러스)** 를 쓰고, 결과는 **에어서치(AiRS) 기반 스마트블록**으로 노출됩니다.

| 요소 | 의미 | 실무 적용 |
|---|---|---|
| **C-Rank (Context)** | 출처·인용의 신뢰도 | 공식 출처(국세청, 홈택스, 한국은행, 통계청) 링크·명시 |
| **C-Rank (Chain)** | 주제 일관성 | 한 블로그는 **금융/세금 한 장르로 고정** (잡블로그 ✗) |
| **C-Rank (Creation)** | 활동성 | 주 1~2회 **일정한 간격** 발행 |
| **DIA+ (정보성·전문성)** | 문서 자체의 깊이·구조 | 단계별 계산·표·근거·정의가 갖춰진 '문제 해결형' 문서 |
| **DIA+ (경험·진정성)** | 작성자의 실제 후기 | **1인칭 경험담 삽입** (→ 섹션 7) |
| **DIA+ (체류·반응)** | 독자 만족 신호 | 목차·이미지·표로 끝까지 읽게 + 저장·댓글 유도 |

### 스마트블록 & 에어서치 대응 (★ 노출 구조 핵심)
요즘 네이버 통합검색은 단일 순위가 아니라 **주제별 블록(스마트블록)** 으로 묶여 노출됩니다. 같은 키워드라도 "○○ 방법", "○○ 후기", "○○ 비용" 등 **세부 의도 블록**이 따로 생깁니다.
- **한 포스팅 = 한 세부 의도**에 정조준 (제목·소제목을 그 의도에 맞춤). 예: "양도세 계산"(정보) vs "양도세 직접 계산 후기"(경험) 블록은 다름
- 스마트블록은 **롱테일·구체 질문**일수록 경쟁이 적고 진입이 쉬움 → 광범위 키워드보다 **시나리오형 롱테일**을 노려라 (섹션 2 참고)
- 에어서치는 개인화·맥락 기반이라, **주제 일관성(C-Rank Chain)** 이 높은 블로그가 관련 블록에 더 잘 뜸

### 2025~2026년 알고리즘 변화 포인트
- **AI 콘텐츠 탐지 강화**: 순수 AI 생성 글은 저품질 위험. 반드시 **개인 경험·수치·감정**의 인간 레이어를 씌울 것 (→ 섹션 7·9)
- **홈판(홈피드) 노출 부상**: 검색 외에 홈 피드가 방문자 핵심 채널. 저장·공유·댓글이 홈판 노출에 영향
- **체류시간·재방문 가중 증가**: 끝까지 읽고 저장하는 글이 가점
- **#내돈내산 #직접계산 진성 후기 우대**: 경험 기반 문서가 노출 우선순위 상승

### 주의사항 (요약 — 상세는 섹션 9 저품질 회피)
- **외부 링크 남발 금지**: 한 포스팅당 Tooly 링크는 **1~2개**만, 후반부 맥락 속 텍스트 링크로 (Tooly 운영 최대 저품질 리스크 → 섹션 9)
- **복붙·자기복제 금지**: 각 포스팅 최소 70% original, 시리즈 글끼리도 구조·문장 차별화
- **키워드 스터핑 금지**: 핵심 키워드 본문 5~7회 이내
- **이미지 5~10장**: 본인 캡처(Tooly 결과) 위주, 파일명·대체텍스트에 키워드
- **제목에 숫자·조건·의도어**: "2026년", "12억 초과", "직접 계산"
- **AI 단독 작성 금지**: 개인 경험·감정·구체 상황으로 인간 레이어 (→ 섹션 7·8)

## 2. 주제 선정 원칙

### 좋은 주제 = 아래 4가지 교집합
1. **검색 수요**: 네이버 검색량 있는 질문형 키워드
2. **Tooly 연결성**: Tooly 계산기로 답변 가능
3. **해설 가치**: 계산기만으론 부족한 맥락·예시·팁 제공 필요
4. **롱테일·저경쟁 (★ 신규)**: 거인(은행·언론사·대형 정보블로그)이 얕게 다루거나 무시하는 **구체적 시나리오** 키워드. 광범위 키워드(예: "양도세 계산")는 상위가 이미 포화 → **"1세대 1주택 12억 초과 양도세 얼마"** 처럼 좁히면 스마트블록 진입이 쉽고, 마침 Tooly 깊이 페이지와 1:1로 연결됨

> **원칙**: 헤드 키워드(연봉 실수령액 등)는 진입장벽이 높다. **롱테일 시나리오 1개를 깊게** 쓰는 글이, 광범위 키워드를 얕게 쓰는 글보다 노출·전환·저품질 회피 모두에서 유리하다.

### 키워드 리서치 도구
- 네이버 광고 **키워드 도구** (검색량 확인, 무료)

- **블랙키위** / **키워드마스터** (네이버 특화)

## 4. 작성 체크리스트

발행 전 반드시 확인:

### 제목
- [ ] 30자 내외 (너무 길면 검색 결과에서 잘림)
- [ ] 핵심 키워드 1개 + 수식어(년도/숫자/질문)
- [ ] 클릭 유도하는 질문형/숫자형
- [ ] 낚시성 금지 (예: "충격", "대박" 등)

### 본문
- [ ] 1,500자 이상 (체류시간 확보)
- [ ] 이미지 5장 이상 (결과 스크린샷·표·그래프)
- [ ] 목차(H2) 3개 이상
- [ ] 표(테이블) 1개 이상
- [ ] 공식/수치에 **출처** 명시
- [ ] Tooly 외부링크 **1~2개** (과다 금지)
- [ ] **경험담 섹션 포함** (1인칭, 구체적 상황·수치·감정 서술)
- [ ] AI 단독 생성 티 나는 표현 제거 ("~하겠습니다", "~살펴보겠습니다" 남발 ✗)

### SEO
- [ ] **제목 공식**: `[세부 키워드] + [숫자/조건] + [의도어]` — 예: "1세대1주택 12억 초과 양도세 직접 계산" (의도어=계산/후기/비교/방법)
- [ ] 대표 키워드가 제목·첫 문단(첫 3줄)·소제목 1개·마지막 문단에 **자연스럽게** 배치

- [ ] 연관 키워드 3~5개를 본문에 분산 (자동완성·연관검색어에서 수집)

## 7. 경험담(체험기) 작성 원칙 ★ 신규

### 왜 경험담이 중요한가

네이버 DIA 알고리즘은 "사용자들의 선호" 또는 "작성자의 후기"처럼 **실제 경험이 담긴 문서**에 높은 점수를 부여합니다. 2025~2026년부터 AI 콘텐츠 탐지가 강화되면서, 개인 경험이 없는 정보 나열 글은 저품질로 판정될 위험이 커졌습니다.

핵심 원칙: **서툴러도 진솔한 경험 > 완벽하지만 AI 냄새 나는 글**

---

### 경험담 작성 3원칙

**① 구체적 수치와 상황을 넣어라**

나쁜 예:
> "연봉 계산기를 사용해보니 편리했습니다."

좋은 예:
> "연봉 협상이 끝나고 '그래서 실제로 얼마 받는 거지?' 싶어서 계산기를 돌려봤는데, 5,200만원에서 실수령액이 360만원대가 나왔을 때 솔직히 조금 실망했어요. 생각보다 많이 빠지더라고요. 그나마 식대 비과세 20만원을 빼고 나서 계산하니까 15만원 정도 더 나왔는데, 이 부분은 처음에 몰라서 헤맸습니다."

---

**② 실패·불편함·헷갈렸던 부분을 솔직하게 써라**

완벽한 후기는 광고처럼 보입니다. 알고리즘도, 독자도 신뢰하지 않습니다.

예시 표현:
- "처음엔 어디에 숫자를 넣어야 하는지 헷갈렸어요"
- "원리금균등이랑 원금균등이 뭔지 몰라서 두 번 검색했습니다"
- "생각보다 이자가 많이 나와서 당황했는데"
- "3개월 전에 알았더라면 다른 선택을 했을 것 같아서 아쉬웠어요"

---

**③ 감정·맥락·결과를 3단 구조로 써라**

```
[상황/배경] → [행동/경험] → [결과/느낌]
```

예시:
> 전세 계약 앞두고 '이 보증금이면 그냥 월세 사는 게 낫지 않나?' 진지하게 고민했어요. (상황)
> 전월세 전환 계산기에 보증금 3억, 전환율 4.5%로 돌려봤더니 월세 환산액이 112만원이 나왔고요. (행동)
> 주변 월세 시세가 85~90만원이라 전세가 유리하다는 결론이 나왔고 덕분에 고민이 정리됐습니다. (결과)

---

### 주제별 경험담 삽입 가이드

| 주제 | 경험담 소재 예시 |
|---|---|
| 연봉 실수령액 | 연봉 협상 후 통장 확인했을 때 느낌, 첫 월급명세서 보고 당황한 경험 |
| 주택대출 | 대출 상담 전 예상 납입액 계산해본 경험, 금리 1% 차이가 생각보다 크다는 걸 깨달은 순간 |
| 복리 계산 | 노후 준비 늦게 시작한 걸 후회하며 시뮬레이션 돌려본 경험 |
| 전세 vs 월세 | 이사 결정 전 직접 계산해서 결정한 과정 |
| 퇴직금 | 이직 시 퇴직금 예상액 몰라서 협상에서 손해 봤던 경험 |
| BMI | 검진 결과 보고 정상 범위인지 헷갈려서 찾아본 경험 |

---

### 경험담 분량과 위치

- **분량**: 200~400자 (너무 길면 지루함. 핵심 상황 1개만 집중)
- **위치**: 정보 설명 섹션 직후, 계산기 CTA 직전에 삽입 (가장 자연스러운 흐름)
- **어투**: 구어체 허용 ("~했어요", "~더라고요", "~인 것 같아요") — 격식체보다 친근한 어투가 체류시간에 유리

---

### 경험담 작성 시 피해야 할 것

| 피할 것 | 이유 |
|---|---|
| "정말 좋았습니다. 강추합니다." | 광고 느낌, DIA 신뢰도 ↓ |
| 경험 없이 "사용하면 좋을 것 같습니다" | 추측성 표현, AI 판정 위험 |
| 수치 없는 막연한 서술 | "많이", "크게" 대신 "30만원", "3개월" |
| 100% 긍정적인 후기 | 단점·헷갈린 점 1개 이상 포함 |

## 8. AI 저품질 판정 회피 체크리스트

네이버는 2025년부터 AI 생성 콘텐츠 탐지 로직을 강화했습니다. 다음 항목을 확인하세요.

- [ ] 1인칭 경험 서술이 최소 1섹션 포함되어 있는가
- [ ] "~하겠습니다", "~살펴보겠습니다" 같은 AI 투의 격식체가 남발되지 않았는가
- [ ] 구체적 수치(금액, 날짜, 기간)가 경험담 안에 들어 있는가
- [ ] 실패·불편·헷갈림 같은 '부정적 뉘앙스'가 최소 1개 포함되어 있는가
- [ ] 본인의 의견·판단이 명확하게 드러나 있는가 ("저는 원금균등이 더 맞는 것 같더라고요")

## 9. 저품질 회피 종합 가이드 ★ 신규

네이버 "저품질"은 글이 **통합검색에서 누락되거나 노출이 급감**하는 상태입니다. 한 번 빠지면 회복이 어렵고, 블로그 전체 지수에 전염됩니다. AI 탐지(섹션 8)는 그 일부일 뿐, 아래 트리거 전체를 관리해야 합니다.

### 주요 저품질 트리거 & 회피법

| 트리거 | 왜 위험 | 회피법 |
|---|---|---|
| **외부링크 과다·상업성** | 네이버는 외부로 트래픽을 보내는·광고성 블로그를 싫어함. **Tooly 운영에서 가장 큰 리스크** | 포스팅당 Tooly 링크 **1~2개**, 본문 후반(정보 제공 후)에만. 단축URL·자동요약카드 남발 ✗, **맥락 있는 텍스트 링크**로 |
| **키워드 스터핑** | 같은 키워드 억지 반복 = 어뷰징 판정 | 핵심 키워드 본문 **5~7회 이내**, 제목에 1회. 동의어·연관어로 분산 |
| **유사문서/자기복제** | 시리즈 글끼리 구조·문장이 너무 비슷하면 중복 판정 | 글마다 **다른 시나리오·다른 경험담·다른 표**. 템플릿은 골격만 재사용, 문장은 새로 |
| **얇은 콘텐츠** | 짧고 정보성 낮은 글 | 본문 **1,500자+**, 표·계산예시·출처 필수 |
| **불규칙·대량 발행** | 며칠 쉬다 하루 5개 = 어뷰징 패턴 | **주 1~2회 일정 간격**. 예약발행으로 리듬 유지 |
| **이미지 도용** | 타 사이트 이미지 펌 = 저품질·저작권 | **본인 캡처(Tooly 결과 화면)** 위주, 외부 이미지 출처 표기 |
| **발행 직후 잦은 수정** | 초기 색인 단계 반복 수정은 불안정 신호로 해석되곤 함 | 발행 전 완성. 수정은 **최소화**하고 며칠 뒤 한 번에 |
| **낚시·금지어** | "충격", "○○충", 과장·비방 | 제목·본문에서 배제 |

### 외부링크 안전 운용 (Tooly 전용 ★)
이 블로그의 목적이 Tooly 유도지만, **그게 곧 저품질의 가장 흔한 원인**이라 균형이 핵심입니다.
- **위치**: 글의 정보 가치를 충분히 전달한 **후반부**(계산 예시·경험담 뒤)에만 배치. 첫 문단·중반 상단 링크 ✗
- **개수**: 포스팅당 1~2개. 같은 URL 반복 노출 ✗
- **형태**: "여기 클릭"이 아니라 **맥락 문장 속 텍스트 링크**("직접 숫자를 바꿔보고 싶다면 → 양도소득세 계산기"). 자동 링크요약 카드 연속 삽입 ✗
- **비율 감각**: 본문 대비 링크·홍보 비중이 작아야 함. 글의 90%는 **독자에게 주는 정보**, Tooly는 거들 뿐
- **현실 인식**: 어차피 nofollow라 SEO 권위는 안 넘어감. 무리해서 링크를 늘릴 이유가 없음 → **자연스러움 > 링크 수**

### 저품질 자가진단 (2주 후)
- 네이버 검색창에 `site:블로그주소` 또는 **글 제목 그대로** 검색 → 내 글이 안 뜨면 누락 의심
- 블로그 통계에서 **검색 유입이 0에 수렴**하면 신호
- 의심되면: 신규 발행을 **양질 1편/주로 줄이고**, 외부링크·키워드 밀도를 더 낮춰 신뢰 회복 (수 주 소요)
